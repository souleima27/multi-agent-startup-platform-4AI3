# Sample Documents Structure

Place your client files here when running `python -m app.run.run_case sample_data/request_demo.json`.

Recommended structure:

- sample_data/documents/statuts/
- sample_data/documents/rc/
- sample_data/documents/if/
- sample_data/documents/cin/
- sample_data/documents/attestation_bancaire/
- sample_data/documents/pitch_deck/

Supported formats:

- Images: .png .jpg .jpeg .bmp .tif .tiff .webp
- PDF: .pdf
- Word: .docx
- PowerPoint: .pptx
